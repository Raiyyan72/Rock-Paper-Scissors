# 🎮 Rock-Paper-Scissors Game

Welcome to the **Rock-Paper-Scissors Game**!  
A simple, interactive, and fun game built with **HTML, CSS, and JavaScript** where you can challenge the computer and test your luck and strategy! 🥳

---

## 🚀 Features

- **Play against the computer** – Classic Rock, Paper, Scissors gameplay.
- **Scoreboard** – Keep track of your wins and the computer's wins.
- **Sound Effects** – Win, Lose, Draw, and Click sounds for an immersive experience 🎵.
- **Background Music** – Toggle music on/off while playing.
- **Start & Exit Screen** – Smooth transitions between pages.
- **Reset Game** – Reset scores anytime to start fresh 🔄.
- **Responsive Design** – Looks great on **mobile, tablet, and desktop** devices.
- **Color-coded Feedback** – Win = green, Lose = red, Draw = light blue 🎨.
- **Animated Buttons** – Hover and click effects for better interactivity.

---



## ⚡ How to Play

1. Click **Start Game** on the opening screen.  
2. Choose **Rock, Paper, or Scissors**.  
3. Watch the computer make its choice.  
4. See the **result message** and **score update**.  
5. Use the **Reset button** to restart the game anytime.  
6. Use **Exit** to go back to the start screen.  
7. Toggle **Music** button to play/pause background music.  

---

## 💻 Tech Stack

- **HTML5** – Structure of the game  
- **CSS3** – Styling, layout, and animations  
- **JavaScript (ES6)** – Game logic, DOM manipulation, and event handling  

---

## 🏆 Fun Tips

- Try to **predict the computer's choice** and win multiple rounds.  
- Listen to the **background music** for a more immersive experience.  
- Play on **different devices** and see how responsive the game is!
